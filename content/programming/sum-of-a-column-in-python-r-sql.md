## Mean of a column in SQL:

```SQL
SELECT SUM(Degree_Length)
FROM Degree
```

## Mean of a column in Python:

```Python
degree['degree_length'].sum()
```

## Mean of a column in R:

```C
sum(degree$degree_length, na.rm = TRUE)
```

<strong>Output:</strong>

```C
42
```
